# CV Generator - Spring Boot

## Project Overview

CV Generator is a Spring Boot REST API project that allows users to manage resume details. It supports creating, retrieving, updating, and deleting resume information using RESTful APIs.

---

## Technologies Used

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Maven
- IntelliJ IDEA
- Git & GitHub

---

## Features

- Create Resume
- Get All Resumes
- Get Resume by ID
- Update Resume
- Delete Resume
- Exception Handling
- REST API Architecture

---

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/resume` | Create Resume |
| GET | `/resume` | Get All Resumes |
| GET | `/resume/{id}` | Get Resume By ID |
| PUT | `/resume/{id}` | Update Resume |
| DELETE | `/resume/{id}` | Delete Resume |

---

## Database

- MySQL Database
- JPA Repository
- Auto CRUD Operations

---

## How to Run

1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Configure MySQL in `application.properties`
4. Run the Spring Boot application
5. Test APIs using Postman
